These orbs have an 80% chance to be dropped when an enemy dies by a player.

| Type      | Affected by ability | Description                                                                     | Value                           | Image |
| --------- | ------------------- | ------------------------------------------------------------------------------- | ------------------------------- | ----- |
| Munny     | Jackpot             | Increases munny when picked up.<br>Drive converter reduces the amount obtained. | 5 - (20% of the entity max HP). |       |
| HP orb    | Jackpot             | Restores health when picked up.                                                 | 10% - 20% of the entity max HP. |       |
| MP orb    | Jackpot             | Restores MP when picked up.                                                     | 10% - 20% of the entity max HP. |       |
| Drive orb | Drive Converter     | Restores drive gauge when picked up.<br>Master form gets xp based on the value. | 10% - 25% of the entity max HP. |       |
| Focus orb | Focus Converter     | Restores focus gauge when picked up.                                            | 10% - 25% of the entity max HP. |       |
>[!tip] Munny value example
>If you kill a zombie it's munny value should range between 5 and a 20% of it's max HP, if it has 100 max HP it's 5 - 20.

>[!tip] HP orb value example
>If you kill a zombie it's HP orb value should range between 10% and a 20% of it's max HP, if it has 100 max HP it's 10 - 20.
