Gummi Blocks are the most common blocks used in Gummi ships.
They are obtained by they come in 16 tint variants and 8 shapes.

| Name         | Description                                                                                                                     | Crafting                                                        |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------- |
| Gummi Block  | They are the basic form of gummi blocks, plain blocks offering basic levels of armor.<br><br>They can also be tinted like this. | ![gummi block recipe.png](/img/user/gummi%20block%20recipe.png)![gummi block tint recipe.png](/img/user/gummi%20block%20tint%20recipe.png)     |
| Gummi Shell  | They are medium blocks offering a bit more armor than the basic ones.                                                           | ![gummi shell recipe.png](/img/user/gummi%20shell%20recipe.png)<br>![gummi shell tint recipe.png](/img/user/gummi%20shell%20tint%20recipe.png) |
| Gummi Dispel | The strongest variant of gummi blocks, offering more armor.                                                                     | ![gummi dispel recipe.png](/img/user/gummi%20dispel%20recipe.png)![gummi dispel tint recipe.png](/img/user/gummi%20dispel%20tint%20recipe.png)   |

**Core**:
The Gummi Core block allows you to be able to assemble a gummi ship.
It's the central part where its health and fuel level are stored when it's in edit mode.
It can be crafted like this:
![Gummi Core recipe.png](/img/user/Gummi%20Core%20recipe.png)

**Shapes**:
Any of the three type of Gummi blocks can be placed in a stonecutter to modify their shape.
![gummi block shapes.png](/img/user/gummi%20block%20shapes.png)
Some shapes have special placement properties, once holding in your hand they will change the block placement outline to add a guide.
![gummi edge placement.png](/img/user/gummi%20edge%20placement.png)![gummi corner placement.png](/img/user/gummi%20corner%20placement.png)
