The pedestal allows you to display the keyblades and other items you have.
![pedestal display.png](/img/user/_img/pedestal%20display.png)

>[!warn] Multiple pedestals
>Setting multiple pedestals for showing keyblades can hinder performance once they are being rendered.

You can customize a few settings:
![pedestal gui.png](/img/user/_img/pedestal%20gui.png)


| Option         | Effect                                                                           |
| -------------- | -------------------------------------------------------------------------------- |
| Pause          | Stops all animation.                                                             |
| Flip           | Turns the item upside down.                                                      |
| Scale          | The size of the item.                                                            |
| Height         | Y position of the item.                                                          |
| Rotation speed | Speed at which the item rotates counterclockwise, can be negative for clockwise. |
| Bob speed      | Speed at which the item bounces up and down.                                     |
| Reset          | Restores all values to default                                                   |
