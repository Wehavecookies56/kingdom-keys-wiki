The Savepoint is a craftable block which allows the player to be healed as well as setting the respawn point.
![save point.png|188](/img/user/_img/save%20point.png)
It can be upgraded with different materials based on the common config value:

| Upgrade | Synthesis material  |
| ------- | ------------------- |
| HP      | Orichalcum          |
| MP      | Illusory Crystal    |
| Hunger  | Hungry Crystal      |
| Focus   | Remembrance Crystal |
| Drive   | Evanescent Crystal  |
| Tier    | Orichalcum Plus     |

Upgrading tiers allows the Savepoint to become a Linked Savepoint, it allows to open an interface with images from all the Savepoints from the same dimension visited by the player and teleport to them.
![linked point.png|135](/img/user/_img/linked%20point.png)

![save point gui.png](/img/user/_img/save%20point%20gui.png)
Upgrading it further beyond to a Warp Point shows all the savepoints regardless of the dimension they are in.
![warp point.png|170](/img/user/_img/warp%20point.png)
